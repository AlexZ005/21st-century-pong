The Pong game
=============

Pong clone written in C++ and using SDL 2.0

Install
=======

## Ubuntu

Just run ``make`` and then ``./pong``.

## Mac

You need to have [brew](http://brew.sh/) installed.
After that, just run the script ``./install.mac.sh``.

## Windows

Not yet tested.
